Overview

This class collects analytics for a publisher’s application.

    Reports the Application Custom Goals to InMobi Ad Tracker. This is a non blocking API (returns immediately), and performs the reporting job in the background. reportCustomGoal:

Reports the Application Custom Goals to InMobi Ad Tracker. This is a non blocking API (returns immediately), and performs the reporting job in the background. When a goal is reported, it is made sure that it reaches InMobi Ad Tracker server whenever the device is connected to internet. Different goals can be reported without worrying about the internet connection.
