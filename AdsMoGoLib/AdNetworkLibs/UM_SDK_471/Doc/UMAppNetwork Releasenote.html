版本：v3.5.1

  • 发布日期：2012.09.17

  • 主要更新：

    1. 全屏广告样式，新增 “信息提示框” 子样式
    2. banner支持特定格式的url，方便自定义
    3. 小把手内置tableview/gridview，支持网站动态调整
    4. 更新文档及demo，支持对iPhone5的适配
    5. 其它

  • 信息提示框&特定schema URL说明：

     我们支持的特定schema为callback，在符合该规则的广告被点击时，相关delegate方法将被调用，关联信息被返回，可以根据返回信息，执行相应的操作

    示例：您在应用中集成了banner和tableview两种样式，如果您想在某条banner的广告被点击后展示一个tableview样式的广告，则可以将banner的这条广告url定义为：callback://tableview?appkey=xxxx, 则可以根据该广告被点击后的回调，展示对应的信息

版本：v3.5

  • 发布日期：2012.08.15
  • 主要更新：
    1. 新增GridView样式，小把手已默认支持该样式，也支持自定义
    2. 修复上个版本中缓存的一个bug
    3. banner轮播时间支持网站动态调整
    4. 更新文档及demo
    5. 其它

  • 注：

    v3.4.1和v3.4.2版本缓存逻辑[对广告信息的缓存，非广告物料]存在一个缺陷，该bug会在您使用以上两版本SDK，且开启广告位缓
存开关时，在部分样式出现，如果您的应用使用了这两个版本的SDK，建议您别开启缓存开关（默认是关闭的），v3.5已解决该问题，
给您带来的不便深表歉意，多谢支持！
